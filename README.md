# Po8 Ledger App

Rust-based application for Ledger Nano X/S+, implementing the Po8 cryptographic primitives.

## Features

*   **Streaming Signatures**: Memory-optimized signing for large ML-DSA-65 signatures.
*   **Blind Signing Prevention**: Custom parser to display `Recipient` and `Amount` on the device screen before signing.
